# 华芯百科 - AI 智能生涯规划系统

一个基于 AI 的学生生涯规划报告生成系统，为初中至本科阶段的学生提供专业、个性化的升学规划、职业匹配和资源对接服务。

## 功能特点

- 🎯 **智能分析**：基于 DeepSeek AI 深度分析学生性格、兴趣和家庭情况
- 📊 **个性化报告**：生成包含 10 个页面的完整生涯规划报告
- 🎨 **精美设计**：红色渐变主题，专业的视觉设计
- 📄 **PDF 导出**：一键生成高质量 PDF 文档
- ⚡ **快速响应**：30 分钟内完成报告生成

## 报告结构

1. **封面页** - 学生姓名、咨询老师信息、生成日期
2. **机构介绍页** - 机构简介、核心服务、专家团队
3. **测评结果页** - 性格测评结论、人格特质分析
4. **职业适配页** - 职业方向推荐、家庭情况分析
5. **规划建议页** - 核心发展路径、职业规划、家庭配合建议
6. **升学项目页** - 项目推荐列表及详解
7. **时间规划页** - 申请时间表
8. **供应链资源页** - 语言培训、游学研学、实习项目、科研赛事
9. **总结页** - 规划逻辑、核心竞争力、行动建议

## 技术栈

- **前端框架**: Next.js 15 + React 19
- **样式**: Tailwind CSS 4
- **AI 服务**: DeepSeek API
- **PDF 生成**: Puppeteer
- **语言**: TypeScript

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置环境变量

创建 `.env.local` 文件：

```env
DEEPSEEK_API_KEY=your_deepseek_api_key_here
```

### 3. 启动开发服务器

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看应用。

## 使用说明

### 1. 填写学生信息

- 学生基础信息（姓名、年级、成绩、目标学历）
- 测评数据（人格类型、性格优劣势、思维模式、兴趣倾向、学科优势）
- 家庭信息（家庭结构、父母职业、经济预算、家庭资源、诉求）
- 特殊情况（考学问题、特色规划、资源缺乏）
- 咨询老师信息

### 2. 输入 DeepSeek API Key

在页面顶部输入您的 DeepSeek API Key。

### 3. 生成报告

点击"生成报告"按钮，等待 AI 分析并生成内容。

### 4. 预览与下载

报告生成后可预览关键内容，点击"下载 PDF"获取完整报告。

## 注意事项

1. **API Key 安全**：请勿在客户端代码中硬编码 API Key
2. **PDF 生成**：首次生成 PDF 可能较慢（需下载 Chromium）
3. **数据完整性**：确保填写所有必填项以获得最佳生成效果
